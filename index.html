Final Excel template design

Use this structure for real uploads. Comparison against previous periods is calculated automatically from the uploaded time series. Add Weight if you want weighted overall actual and target.

Recommended workbook structure

Sheet 1: Data_Input
Sheet 2: Metric_Settings
Sheet 3: Instructions

Weight (Optional)

Used when combining multiple rows into one KPI. Enter the operational volume behind the metric so larger workloads influence the overall result more than smaller ones.

• SLA → Contact volume
• AHT → Handled contacts
• QA → Number of audits
• Resolution Rate → Cases handled

Leave this column blank if you want the dashboard to use a simple average.

Template rules

Rules designed for a large audience and flexible KPI usage.

✅ Data_Input stores time-series rows only
✅ Metric_Settings stores KPI logic once, not repeated in every row
✅ Unknown KPIs are allowed as long as settings are provided
✅ LOB, Channel, Site and Team are optional but supported
✅ Forecast stays hidden where values are unavailable
✅ Previous comparison is derived internally from the prior period, no extra column needed
✅ Weighted overall actual and target are used when a Weight column is provided
